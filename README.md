# Steam Price Checker
This project is my "First Personal Project" for Boot.dev as my first submission. This is a CLI tool to compare Steam game prices across regions using the Steam Store API using Python. Basically, I chose this idea since I very often check regional prices ever since I've moved abroad. 

## 1. Install Python

Ensure **Python 3.8 or newer** is installed.

Check your version:

```bash
python --version
```

## 2. Install Dependencies

The project requires the **requests** library to communicate with the Steam API and the currency exchange API.

Install it using pip:

```bash
pip install requests
```

---

## 3. Navigate to the Project Folder

Open a terminal and move to the project directory.

Example:

```bash
cd steam-price-checker
```

---

## 4. Run the Program

```bash
PYTHONPATH=src python3 main.py
```
